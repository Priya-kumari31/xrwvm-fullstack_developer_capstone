# XRWVM Full Stack Developer Capstone

## Project Description
This project is a full stack dealership application developed using Django and React.  
The application allows users to view car dealerships, dealer details, and submit reviews.

## Features
- View dealership information
- View dealer details
- Add customer reviews
- Review sentiment feature
- REST API integration
- React frontend with Django backend

## Repository Name
xrwvm-fullstack_developer_capstone

## Technologies Used
- Django
- React.js
- JavaScript
- SQLite Database
- Docker
- Kubernetes

## Project Structure
- `server` - Django backend API
- `frontend` - React frontend application
